# T3-MC949

Adicione a pasta ./videos com os vídeos que estão no Drive.

## Reprodutibilidade

O script do Powershell `run.ps1` deve ser usado para rodar o código, e funcionará no Windows 11 caso esteja instalados o Python (testado na versão 3.13.9) e o Microsoft Visual C++ 2015-2022 (testado na versão 14.44.35211), além de a política de execução estar definida como `Unrestricted`.

Também há os arquivos: `run.bat` - para execução no prompt de comando clássico do Windows, e `run.sh` - para execução no Bash em ambiente Linux, embora esta última opção não ofereça a qualiade de narração desejada.
